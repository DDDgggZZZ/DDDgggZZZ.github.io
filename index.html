04 — Projects

Work I'm proud of.

№ 01 · 2025 Core contributor

Data-Fact-Driven Visual Analysis for Time-Series Generation

A visual analytics system that uses automated fact extraction, multi-objective subset selection, and LLM-assisted interaction to make high-quality, controllable time-series data augmentation tractable. I led the uncertainty visualization component and the generation & evaluation experiments.

Vue.js D3.js Python LLM VA
№ 02 · 2025 Principal Investigator

Tower-AI — Real-time Hazard Detection for Telecom Infrastructure

A provincial-level SIT project I lead. Built a two-stage deep-learning pipeline (cruise detection + precise detection) on top of an improved YOLOv8 for multi-scenario tower inspection — power lines, corrosion, intrusion — with a visualization dashboard for real-time alerts and device management.

YOLOv8 PyTorch CV Vue.js
